WINDOWS 12002 / 12030 FINAL NETWORK FIX — LOADER 1.0.3

Причина:
Предыдущая переделка сети добавила GitHub API, отдельные direct/proxy-подключения, короткие таймауты и дополнительные HTTP-заголовки. На части Windows это приводило к WinHTTP 12002 и 12030 даже при рабочем интернете.

Финальное исправление:
- транспорт WinHTTP возвращён к стабильной схеме из первой рабочей версии;
- используется WINHTTP_ACCESS_TYPE_AUTOMATIC_PROXY;
- возвращены проверенные таймауты 8000 / 8000 / 15000 / 30000;
- удалены Connection: close, принудительный TLS и нестандартные Accept-заголовки;
- GitHub API удалён из цепочки;
- основной источник — raw.githubusercontent.com;
- резервный источник — jsDelivr;
- ошибки 12002, 12030, 12029 и DNS автоматически повторяются один раз через 350 мс;
- HTTP-код, этап сбоя и Windows error сохраняются в сообщении;
- защита manifest остаётся fail-closed.

Установка:
1. Распаковать поверх полного проекта.
2. Полностью удалить build и out/build либо запустить build-release.bat.
3. Запускать только новый build/Release/NeverloseLoader.exe.
