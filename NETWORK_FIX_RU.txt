СЕТЕВОЙ HOTFIX 1.1.1

Изменения:
- исправлено ложное сообщение Request failed (HTTP 0);
- добавлены реальные коды и тексты ошибок WinHTTP;
- принудительно включён TLS 1.2;
- добавлен повторный запрос без системного прокси;
- добавлен резервный источник jsDelivr;
- добавлен локальный кэш последнего корректного манифеста;
- версия лоадера повышена до 1.1.1.

Установка:
1. Скопировать CMakeLists.txt и папку src в ПОЛНЫЙ проект с заменой.
2. Запустить build-release.bat для чистой Release-сборки.
3. Проверить build/Release/NeverloseLoader.exe на нескольких компьютерах.
4. Создать GitHub Release loader-1.1.1 и прикрепить новый EXE.
5. Только после публикации изменить minimumVersion в обоих файлах manifest на 1.1.1 и отправить отдельным коммитом.

До публикации Release minimumVersion намеренно оставлен 1.1.0.
